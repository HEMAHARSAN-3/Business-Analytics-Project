# 📊 Business Performance & KPI Analytics Dashboard

## 🚀 Project Overview

This project is an end-to-end Business Analytics solution designed to analyze sales performance, track key performance indicators (KPIs), and generate actionable insights using SQL and Tableau.

The dashboard simulates real-world business intelligence systems used by companies to monitor performance and make data-driven decisions.

---

## 🎯 Objectives

* Analyze sales performance across regions and categories
* Track key business KPIs (Revenue, Orders, Profit)
* Identify trends and growth patterns
* Provide actionable business insights using dashboards

---

## 🧰 Tech Stack

* SQL (SQLite)
* Python (Pandas, Matplotlib, Seaborn)
* Tableau
* Excel / CSV

---
